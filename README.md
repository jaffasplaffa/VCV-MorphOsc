


# VCV-MorphOsc module by Jaffasplafa (Jakob Skouborg)

This is an oscillator that creates the basic waveforms saw, sine and triangle and let's 
you morph between them, for all kinds of exotic waveforms. 

# Parameters
Pitch,
Breakpoint,
Rise & Fall,
Sin/Tri,
